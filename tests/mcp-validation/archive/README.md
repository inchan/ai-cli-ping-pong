# 아카이브된 문서

검증 과정의 역사적 맥락을 보존하기 위한 문서 저장소입니다.

## 디렉토리 구조

### phases/ - Phase별 원본 보고서
- `VALIDATION_HISTORY.md`로 통합되었으나, 상세 기술 정보 보존을 위해 원본 유지
- 각 Phase의 전체 컨텍스트와 코드 예제 포함
- Phase 0: MCP SDK 설치
- Phase 1: 환경 준비 및 기본 검증
- Phase 3: 행동 테스트
- Phase 4: 메트릭 수집 및 문서화

### testing/ - 테스트 가이드 원본
- `TESTING_GUIDE.md`로 통합되었으나, 전체 체크리스트 보존을 위해 원본 유지
- 상세한 수동 테스트 절차 및 100개 이상의 체크리스트 포함
- MCP Inspector 가이드
- 수동 테스트 체크리스트

### obsolete/ - 대체되거나 중복된 문서
- 다른 문서로 요약되거나 대체된 문서들
- 역사적 기록 목적으로만 보존

## 사용 지침

**우선 참조**: 루트 디렉토리의 통합 문서
- `VALIDATION_HISTORY.md` - Phase별 검증 이력 통합
- `TESTING_GUIDE.md` - 테스트 가이드 통합

**상세 참조**: 이 디렉토리의 원본 문서들
- 코드 예제, 전체 체크리스트 등 상세 정보 필요 시
